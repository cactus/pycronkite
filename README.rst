pycronkite
===========

.. contents::

Requirements
------------

Build requirements:

- gcc
- libcronkite
- python

Run requirements:

- libcronkite
- python


Building
--------

::

    $ cd cmodule
    $ python setup.py build


Running Tests
-------------

To run the tests, do the following after the build phase::

    $ python setup.py test


Installing
----------

After building, while in the build directory::

    $ python setup.py install
